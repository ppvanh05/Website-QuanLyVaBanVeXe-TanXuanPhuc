Bước 1: Mở terminal trong VS Code

Ví dụ mở thư mục muốn chứa project rồi mở terminal.

Bước 2: Clone repo

Chạy:

git clone https://github.com/ppvanh05/kien-tap-code.git

Git sẽ tải toàn bộ project xuống máy.

Bước 3: Đi vào project Angular của bạn

cd kien-tap-code/ANGULAR/kien-tap

Bước 4: Cài thư viện

npm install

Bước 5: Chạy Angular

ng serve

Xong mở:

http://localhost:4200

để chạy : chuột phải chọn open in....

ng serve

click localhost

Sau khi code xong để đẩy lên GitHub:

git add .

git commit -m "update giao dien"

git push
